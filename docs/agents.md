# Agentes Nox

El ecosistema Nox se organiza mediante agentes con responsabilidades separadas.

## Nox Central

Nox Central coordina el sistema. Define prioridades, valida tickets, conserva decisiones y mantiene la coherencia general del ecosistema.

Responsabilidades:

- Organizar proyectos.
- Validar tickets.
- Centralizar decisiones.
- Mantener la documentacion operativa.
- Detectar riesgos de alcance o coherencia.

## Antigravity

Antigravity es el espacio de exploracion y diseno. Convierte ideas en propuestas, sistemas y tickets preparados para validacion.

Responsabilidades:

- Explorar ideas.
- Disenar experiencias y metodologias.
- Proponer estructuras.
- Convertir trabajo ambiguo en tickets claros.

## Codex

Codex es el agente de implementacion. Ejecuta tickets aprobados con cambios pequenos, controlados y documentados.

Responsabilidades:

- Leer tickets.
- Revisar repositorio y documentacion.
- Implementar solo el alcance aprobado.
- Validar resultados.
- Reportar cambios, riesgos y estado de GitHub.

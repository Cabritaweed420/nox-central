# Vision Nox

Nox existe para mantener claridad, organizacion y disciplina en proyectos que crecen con el tiempo.

## Principios

- Claridad: cada decision debe poder entenderse sin depender de memoria informal.
- Organizacion: la informacion debe tener un lugar definido.
- Disciplina: el flujo de trabajo debe respetarse incluso en tareas pequenas.
- Sistemas: el proyecto debe apoyarse en procesos repetibles.
- Escalabilidad: la estructura debe crecer sin volverse caotica.
- Estabilidad: los cambios deben ser seguros, trazables y validables.
- Documentacion: lo importante debe quedar escrito.

## Proposito

Este repositorio centraliza la informacion necesaria para coordinar agentes, tickets, proyectos y decisiones del ecosistema Nox.
